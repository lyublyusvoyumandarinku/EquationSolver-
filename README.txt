EquationSolver

Этот проект представляет собой программу, позволяющую пользователю получить решение квадратного уравнения. 
Для запуска программы на устройстве должен быть установлен компилятор g++ (MinGW и т д) или любая среда разработки, 
поддерживающая язык C. Чтобы запустить проект, необходимо скомпилировать файлы проекта с расширением .cpp 
с помощью командной строки (например, так: g++ main.cpp func.cpp test.coo -o solve.exe где solve.exe - исполняемый файл, 
который после этого следует запустить. 
Программа предложит пользователю ввести коэффициенты уравнения (a, b, c). Если ввод некорректен, программа 
укажет на это и предложит повторить его. После выполнения выдаст результат: количество корней уравнения и
сами корни при их наличии, а также уравнение в стандартном виде. 
В программу встроена возможность тестирования на различных входных данных (чтобы запустить тестирования, в командной строке необходимо прописать define TESTS).
Если какой-либо тест не пройден, программа выведен все вхоные и выходные данные непройденного теста.
Лицензия отсутствует, разрешается печать, копирование и использование в некомерческих целях.
Версия: EquationSolver 1.0 (с) Колянчик 08.2022
